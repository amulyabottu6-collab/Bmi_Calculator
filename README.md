# Bmi_Calculator
# ⚖️ BMI Calculator – Python Tkinter Project
## 📌 Project Overview
This is a simple GUI-based BMI (Body Mass Index) Calculator built using Python Tkinter. It takes user input for weight and height, calculates BMI, and displays the health category based on the result.
---
## 🚀 Features
- User-friendly graphical interface
- Takes weight (kg) and height (m) as input
- Calculates BMI instantly
- Displays BMI value with category
- Shows health status:
  - Underweight
  - Normal weight
  - Overweight
  - Obese
- Input validation for correct values
- Error handling using popup messages
---
## 🛠️ Technologies Used
- Python
- Tkinter (GUI)
- Messagebox (for error alerts)
---
## ▶️ Formula Used
BMI = weight / (height × height)
---
## ▶️ How to Run
Install required library (Tkinter is built-in with Python):
Run the program:
python bmi_calculator.py
---
## 📊 Output
Enter weight (kg) and height (m), then click **Calculate BMI** to see:
- BMI value
- Health category (Underweight / Normal / Overweight / Obese)
---
## ⚠️ Error Handling
- Shows error if weight or height is empty
- Shows error if values are negative or zero
- Shows error for invalid input (non-numeric values)
